# PIA Flight System

A Python-based flight management system for PIA with MySQL integration. This system allows users to manage flights, passengers, bookings, and ticket generation.

## Installation

1. Clone this repository.
2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3. Set up MySQL database as per `pia_flight_system.sql`.
4. Run the application.

## Usage

1. Configure your MySQL credentials in the `Database` class.
2. Run the `main.py` file to interact with the flight system.
